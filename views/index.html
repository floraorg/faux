faux

a minimal, fast, and open source placeholder image service

examples:

/300/150/b34fc7?r=10&d=true&t=true

/150/150/435eee?r=20&g=true

usage:

/:width/:height # default gray

/:width/:height/:color # custom color (hex)

/:width/:height/:color?r=N # rounded corners

/:width/:height/:color?d=true # dotted bg

/:width/:height/:color?g=true # gradient bg

/:width/:height/:color?t=true # add dimensions
